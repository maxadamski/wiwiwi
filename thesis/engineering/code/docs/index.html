Welcome!

Colosseum is a free online platform for automatic evaluation of AI agents in games.

Features

  • Web interface for user submissions and administration
  • Agents and new games can be written in different programming languages
  • Optional self-hosted installation for fast prototyping
  • Execution environment with anti-cheat protection
